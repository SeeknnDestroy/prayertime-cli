# Security Policy

## Supported Versions

Until `v1.0.0`, only the latest tagged release is considered supported.

## Reporting

Please report security issues privately through GitHub Security Advisories or direct maintainer contact instead of opening a public issue.

## Privacy

`prayertime-cli` is designed to avoid telemetry, trackers, and advertising SDKs. MVP 1 uses public HTTP APIs for geocoding and prayer-time retrieval and does not require API keys.
